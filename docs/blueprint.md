# **App Name**: CareerCrafter AI

## Core Features:

- User Authentication & Profile: Secure user registration, login, and basic profile management with data stored in MongoDB.
- Resume Upload & Storage: Allow users to securely upload and store multiple resumes (PDF/DOCX) for analysis, saving metadata in MongoDB.
- AI Resume Optimization Tool: A generative AI tool that analyzes uploaded resumes to identify weaknesses (e.g., weak summaries, lack of measurable achievements, missing keywords) and suggests specific, actionable improvements for optimization.
- ATS Compatibility Scoring: Utilize AI to evaluate the resume's Applicant Tracking System (ATS) readiness, providing a score based on keyword optimization, formatting, achievements, and section completeness.
- AI Job Description Match Tool: A generative AI tool that compares an uploaded resume against a provided job description to calculate a match percentage, highlighting both matched and missing skills.
- Personalized Analytics Dashboard: An interactive dashboard displaying all analyzed resume insights, including ATS scores, improvement suggestions, and job match results in an intuitive format.
- Detailed Report Generation: Generate a downloadable PDF report containing a comprehensive summary of the resume analysis, ATS score, job match results, and all improvement suggestions.

## Style Guidelines:

- Primary brand color: A deep, professional indigo blue (#3E2999) conveying insight and guidance, chosen for its professional yet engaging tone appropriate for career development.
- Background color: A very light, desaturated version of the primary blue (#F2F2F7) to maintain a clean, open, and serene user interface, aligning with a light theme.
- Accent color: A vibrant, clear sky blue (#44A7DB) analogous to the primary, providing clear contrast for interactive elements, calls-to-action, and highlights, symbolizing clarity and opportunity.
- Body and headline font: 'PT Sans' (humanist sans-serif) selected for its modern yet warm appeal, ensuring high readability and a professional personality for all textual content, from analytical reports to user guidance.
- Utilize a consistent set of clean, modern line icons that effectively communicate functionality. Emphasize icons related to documents, scores, progress, and skill indicators for clarity within the analytics.
- Implement a dashboard-centric layout with clearly delineated sections for various analyses (resume strength, ATS, job matching). Use cards, collapsible panels, and intuitive navigation for content organization, ensuring mobile responsiveness.
- Incorporate subtle, functional animations for state changes and user feedback. This includes loading spinners for AI processing, smooth transitions between different dashboard views, and hover effects on interactive elements.
